Variable

Variable (бһўбһҗбҹҒбһҡ) бһӮбһәбһҮбһ¶бһ‘бһёбһҸбһ¶бҹҶбһ„бһҳбһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбҹ”
бһңбһ¶бһўбһ¶бһ…бһ•бҹ’бһ‘бһ»бһҖбһҸбһҳбҹ’бһӣбҹғбһ•бҹ’бһҹбҹҒбһ„бҹ— бһҠбһјбһ…бһҮбһ¶ бһӣбҹҒбһҒ бһўбһҖбҹ’бһҹбһҡ бһ¬ бһҸбһҳбҹ’бһӣбҹғбһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ”

1. бһҡбһ”бҹҖбһ”бһ”бҹ’бһҡбһҖбһ¶бһҹ Variable

бһҡбһ¶бһӣбҹӢ Variable бһҖбҹ’бһ“бһ»бһ„ PHP бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҸбҹ’бһҡбһјбһңбһҸбҹӮбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҠбҹ„бһҷбһҹбһүбҹ’бһүбһ¶бһҠбһ»бһӣбҹ’бһӣбһ¶бһҡ $ бһҮбһ¶бһ“бһ·бһ…бҹ’бһ…бҹ” EX:

<?php
$name = "Heng Oudom"; // бһ”бҹ’бһҡбһ—бҹҒбһ‘бһўбһҸбҹ’бһҗбһ”бһ‘ (String)
$age = 20;           // бһ”бҹ’бһҡбһ—бҹҒбһ‘бһӣбҹҒбһҒ (Integer)
$gpa = 4.5;          // бһ”бҹ’бһҡбһ—бҹҒбһ‘бһӣбҹҒбһҒбһ‘бһҹбһ—бһ¶бһӮ (Float)
?>

2. бһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһҲбҹ’бһҳбҹ„бҹҮ Variable

  1. бһҸбҹ’бһҡбһјбһңбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҠбҹ„бһҷбһҹбһүбҹ’бһүбһ¶ $
  2. бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҹбһүбҹ’бһүбһ¶ $ бһҸбҹ’бһҡбһјбһңбһҸбҹӮбһҮбһ¶ бһўбһҖбҹ’бһҹбһҡ бһ¬бһҹбһүбҹ’бһүбһ¶ Underscore (_). бһҳбһ·бһ“бһўбһ¶бһ…бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҠбҹ„бһҷбһӣбҹҒбһҒбһЎбһҫбһҷбҹ”
  3. бһҲбҹ’бһҳбҹ„бҹҮ Variable бһҠбҹӮбһӣбһҳбһ¶бһ“бһўбһҖбҹ’бһҹбһҡбһҸбһјбһ… бһ“бһ·бһ„бһ’бҹҶ бһӮбһәбһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ $color бһ“бһ·бһ„ $COLOR бһӮбһәбһҮбһ¶ Variable бһ–бһёбһҡбһ•бҹ’бһҹбҹҒбһ„бһӮбҹ’бһ“бһ¶бҹ”
